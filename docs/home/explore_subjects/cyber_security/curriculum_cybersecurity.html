Studying Cyber Security

Structure of studies

In computer science you are going to study in many different subjects. You start with some classic basics like Cryptography and IT Law. After the mandatory theory you will be able to specialize in many different topics that are from the Cyber Security and Computer Science field. You will get more practice-oriented in study project or internships.

In the specialization of Cyber Security you can dive into topics like Ethical Hacking, Network Security or learn about how to secure pervasive systems. You don't have to go through all modules that are available but can choose what is most interesting for you. In the specialization of Computer Science there are also many different subjects like Operating Systems, more technical stuff like Embedded Realtime Systems or more hands-on topics like Software Project Managment. Of course, here you can also choose the subjects that you are most interested in.
